eBay API Settings

You need a free eBay Developer account to get credentials. Visit developer.ebay.com → My Account → create an app for Production keys, then generate a User Token under Get a Token.
⚠ Credentials are saved only in your browser's localStorage. The CORS proxy forwards your request to eBay's API — your token is transmitted to eBay, transiting the proxy. Use your own proxy if you want end-to-end control.
developer.ebay.com → My Account → User Tokens → Get a Token for this Application
